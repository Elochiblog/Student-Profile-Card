# Student Profile Card App

This is a React application that displays reusable student profile cards.

## Features

- Reusable components
- Dynamic badges
- Score progress bars
- Conditional rendering
- Responsive layout

## Technologies Used

- React
- JSX
- CSS
- Vite

## Concepts Practiced

- Props
- Destructuring
- Component composition
- Conditional rendering
- .map()
- .reduce()
